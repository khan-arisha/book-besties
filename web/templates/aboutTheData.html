About the Data

Data source

Dataset Title: NYT Hardcover Fiction Bestsellers: Lists

Dataset DOI

Data Organized by Weeks accessed 5 April 2023.

Data Organized by Title accessed 5 April 2023.

Author: Jordan Pruett Jordan Pruett on Twitter.

Data Scope

This data lists NYT Hardcover bestsellers between 1931 and 2020. The data was scraped from Hawes Publications, an online repository that publishes a pdf of the list for every year.

Data Limitations

The dataset author, Jordan Pruett, makes sure to note that the New York Times Bestseller List reflects editorial choices and does not necessarily reflect what people were reading in those years. Additionally this data only includes hardback books and excludes paperback books.

Data Modifications

The original dataset included the following variables: year, week, rank, best rank, title id, title, author, total weeks, first week, debut rank, and best rank. We modified the dataset to only include best rank, title id, title, author, total weeks, and first week. Additionally, to make searching the data easier, we capitalized all book title and author names.

Citation

Pruett, Jordan, 02/01/2022, “New York Times Hardcover Fiction Bestsellers, 1931–2020”. DOI: https://doi.org/10.18737/CNJV1733p4520220211. Post45 Data Collective, V1.
